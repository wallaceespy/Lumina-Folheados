💎 Lumina Folheados
A Lumina Folheados é uma vitrine digital elegante e moderna desenvolvida para exibir uma coleção exclusiva de semijoias e folheados. O foco do projeto é proporcionar uma experiência de usuário (UX) fluida, destacando a sofisticação das peças através de um design limpo e responsivo.

🚀 Link do Projeto: https://luminafolheados.netlify.app/

✨ Funcionalidades
Catálogo Interativo: Exibição detalhada de produtos com imagens de alta qualidade.

Design Responsivo: Otimizado para dispositivos móveis, tablets e desktops.

Interface Minimalista: Foco total no produto para aumentar a conversão.

Navegação Fluida: Transições suaves e menu intuitivo.

Integração de Contato: Atalhos diretos para facilitar a jornada de compra do cliente.

🛠️ Tecnologias Utilizadas
O projeto foi construído utilizando as melhores práticas de desenvolvimento web moderno:

HTML5 / CSS3: Estrutura e estilização avançada com foco em performance.

JavaScript: Lógica de interatividade e manipulação dinâmica de elementos.

Netlify: Plataforma utilizada para o deploy contínuo e hospedagem escalável.

Google Fonts: Tipografia selecionada para transmitir elegância e legibilidade.

🎨 Layout
O design da Lumina Folheados segue uma paleta de cores sofisticada, utilizando tons que remetem ao luxo e à durabilidade dos metais nobres, garantindo que as peças sejam as protagonistas da tela.

"A beleza está nos detalhes, e nosso site reflete o brilho de cada peça."
